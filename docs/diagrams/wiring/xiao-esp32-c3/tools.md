# Tools

## Main ones

- Soldering iron with temperature control
- Solder (63/37 tin/lead or lead-free, roughly 0.8mm diameter)
- Brass tip cleaner
- Desoldering pump and/or desoldering wick
- Wire strippers
- Isopropyl alcohol and brushes
- Magnifying glass
- Pliers
- Tweezers

## Safety

- Safety glasses with side protection
- Fume extractor/P100 mask or open windows + fan + N95/KF94/FFP2 mask
- Soldering iron stand
- Fire-resistant mat or surface to work on

## Testing

- Multimeter
- Oscilloscope (optional)
- Logic analyzer (optional)
- JTAG debugger (optional)
